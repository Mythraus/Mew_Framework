Helpers

What are helpers?

Helpers are little addons to a page that you can make without actually having to edit the pages source, It isn't really tested yet but it should work

Helper Example

					
					    // The class name must be the helpers class name in the 

					    //database

					    class IndexHelper 

					    {
					    /*

					    *  Every helper must contain a Load function which

					    *  must have a $Database variable and a $Description 

					    *  variable.

					    */ 

					        public function Load($Database, $Description)

					        { 

					            echo "Hello I help the index"; 

					            $Database->query("INSERT INTO notes(note) VALUES('I have helped index')"); 

					            echo $Description; 

					            // And that's how easy it is! 

					        } 

					    }
