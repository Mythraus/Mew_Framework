What is MewFramework?

MewFramework is basic PHP framework that is written using the Object Oriented Pattern, Mew allows for faster creation of small apps.
Mew uses PDO for database access, it comes pre setup for a mysql environment however it can be changed with the right knowledge, PDO was chosen because of it's compatibility with multiple database drivers. Link
Mew also uses RainTPL for templating, this allows php and html to seperated allowing for easy code editing to both the template and the backend.

It's easy to create pages with Mew

All you have to do is create a php file with the name you want in the PAGES folder and it's good to go!
For instance you may want to make a page such as http://www.example.com/MyNewPage all you have to do is create a file called "MyNewPage.php" in the PAGES folder which can be found inside the APPLICATION folder!
